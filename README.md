Welcome to BMGestureScoring!
===================


 这是一个手势打分的小控件,百分百贴合.

核心代码使用开源框架:https://github.com/kevinzhow/PNChart

**需求**

 - 需求是0分不能回退到100.99.98
 - 100分以后不能进到0.1.2.3
 - 但是100分以后可以回退到99.98
